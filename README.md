# Multi-User News Distribution System  

### TECHNOLOGY USED
```
Frondend: Python  
Backend: Python  
Database: MySQL
```    
*Link*: https://github.com/mone-esha/multi-user-news-distribution-system  
### FUNCTIONAL SPECIFICATION  
**User Features:**
  1. Option to add new Users (Name, Age, E-mail and Contact)
  2. Refreshing Users.
  3. There is a option to select User to:  
         a) Update User's info.  
         b) Delete User's info.  
  4. Option to search User by name.
     
**News Features:**
  1. Option to add News (Title, Body, User)
  2. Refreshing News.
  3. There is a option to select News to:  
        a) Update News.  
        b) Delete News.  
  4. Option to search News by title and content.
